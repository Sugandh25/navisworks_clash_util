This is a smple Navisworks button to start an external exe. 
 

Instruction 
----------- 

(1) create a folder under Navisworks install 

e.g., 

C:\Program Files\Autodesk\Navisworks Manage 2015\Plugins\HackathonApp

(2) Copy this content to the above. so it looks like: 

C:\Program Files\Autodesk\Navisworks Manage 2015\Plugins\HackathonApp\AECHackthon.exe
C:\Program Files\Autodesk\Navisworks Manage 2015\Plugins\HackathonApp\HackathonApp.dll


HackathonApp.dll - this is button dll

AEC Hackathon.exe - will be the name of Python exe. (Hard coded for simplicity). 

When Python script is ready, simply replace it with the same.
